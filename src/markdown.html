STATS SOUND Extension Command

Play the selected sound

STATS SOUND
TYPE = DEFAULT* or EXCLAMATION or ASTERISK or HAND or QUESTION
FILE=“sound file”

/HELP

* Required
** Default

HELP displays this help and does nothing else

STATS SOUND TYPE=EXCLAMATION.

This command is only available on Windows.

Specify either TYPE or FILE. If nothing is specified, the Windows Beep sound is produced. FILE must specify a wav file.

Using TYPE, the sound played is whatever is associated with the selected sound according to the settings in the Windows Control Panel Sounds dialog.

If both TYPE and FILE are specified, FILE is played first.
